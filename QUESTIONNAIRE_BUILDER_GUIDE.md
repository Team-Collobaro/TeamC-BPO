# Questionnaire Builder - Admin Guide

## Overview

The Questionnaire Builder allows admins to create quiz questions for any video/module. Learners must answer these questions correctly after watching the video to unlock the next module.

## How to Use

### Step 1: Access Questionnaire Builder

1. Login as **admin**
2. Go to **Admin Panel**
3. Click **"Questionnaire Builder"** in the sidebar

### Step 2: Select Course

- View all available courses
- Click on a course to select it
- The course will be highlighted in blue

### Step 3: Select Video/Module

- View all videos in the selected course
- Each video shows:
  - Module number and title
  - Number of existing questions (if any)
- Click on a video to select it

### Step 4: Build Questions

#### Add Questions:
1. Click **"+ Add Question"** button
2. Enter the question text
3. Fill in 4 answer options
4. **Click the radio button** to mark the correct answer
5. Repeat for more questions

#### Edit Questions:
- Update question text or options anytime
- Change the correct answer by clicking a different radio button
- Click **"Remove"** to delete a question

#### Save Questions:
- Click **"Save Questions"** button at the bottom
- Questions are saved to the module
- Learners will see these questions after watching the video

## How Learners See Questions

1. Learner watches video
2. Marks video as complete
3. **Quiz appears** below the video
4. Learner answers all questions
5. Clicks "Submit Answers"
6. Gets instant feedback (pass/fail)
7. If passed, next module unlocks

## Tips

- **Clear questions**: Write clear, specific questions
- **Avoid ambiguity**: Make sure only one answer is clearly correct
- **Test difficulty**: Balance easy and challenging questions
- **Key concepts**: Focus on important learning points
- **Optimal number**: 3-5 questions per video works well

## Troubleshooting

### "Failed to save questions"
- Make sure all questions have text
- Ensure all options are filled in
- Check that a correct answer is selected
- Try again

### Questions not showing for learners
- Verify questions are saved (green badge on video)
- Check that learner has completed the video
- Refresh the page

### Can't select a video
- Make sure you selected a course first
- Verify the course has modules/videos
- Try refreshing the page
